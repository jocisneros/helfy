# How To: Setup Virtual Environment
## 1) Create Virtual Environment Files

MacOS
```
$ python3 -m venv venv
```

Windows
```
> python -m venv venv
```
## 2) Activate Environment

MacOS
```
$ venv/bin
$ source activate
```

Windows
```
> cd venv/Scripts
> activate
```

## 3) Install Dependencies
Run
```
pip install -r requirements.txt
```
